# ALPHACryptic

ALPHACryptic is a kid-friendly cryptic clue game where players decrypt inspiring phrases by matching numbers to letters.

## Features

- **Multiple Levels**: Challenging levels with inspiring and motivational phrases.
- **Sound Effects**: Interactive audio feedback for inputs, correct answers, mistakes, and victory.
- **Mistake System**: Players have a limited number of mistakes before the board resets.
- **Clean UI**: A focused, ad-free experience designed for kids.
- **Responsive Design**: Playable on both desktop and mobile devices.

## How to Play

1. Tap a tile to select it.
2. Use the on-screen keyboard or your physical keyboard to enter letters.
3. Each number corresponds to a specific letter across the whole phrase.
4. Once you've filled in all the letters, click "Decrypt the whole phrase" to check your answer.
5. If you make 3 mistakes, you'll need to reset the board and try again.

## Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js with Express (for serving static files)
- **Audio**: Web Audio API

## Development

To run the project locally:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open your browser at `http://localhost:3000`
